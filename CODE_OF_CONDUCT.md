# Code of Conduct

## Scientific Integrity

MaxEnt-Kernel is a scientific tool. All interactions in this repository must adhere to the standards of academic integrity:

1. **Cite properly.** If you use this software or its results in any publication, presentation, or report, cite the original work as specified in [CITATION.cff](CITATION.cff).

2. **Report honestly.** If you discover an error in the algorithms, the theory, or the numerical results, report it via GitHub Issues. Honest error reporting strengthens science.

3. **Respect intellectual property.** This is proprietary software with a scientific license available on request. Do not redistribute, re-license, or commercialize without written authorization.

4. **Collaborate respectfully.** Discussions, issues, and any correspondence should remain professional and constructive.

---

## Enforcement

Violations of this code (plagiarism, unauthorized redistribution, abusive communication) will result in:

- Revocation of any granted scientific license
- Reporting to the relevant academic institution
- Legal action under French and EU intellectual property law if warranted
