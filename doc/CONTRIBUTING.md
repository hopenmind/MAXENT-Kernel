This repository does **NOT** accept unsolicited pull requests, code contributions, or feature patches without a prior written agreement.

---

## For Scientific Collaborators

If you are an academic researcher and wish to:

- Extend the solver for a new class of spectral densities
- Implement additional physical models
- Contribute benchmark data from experiments
- Propose corrections to the theory or algorithms

**Process:**

1. Contact us at **contact@hopenmind.com** with a description of your intended contribution
2. We will evaluate the proposal and, if accepted, issue a **Contributor License Agreement (CLA)**
3. The CLA ensures that:
   - All contributed code becomes the intellectual property of Hope 'n Mind SASU - Research
   - You retain co-authorship credit in CITATION.cff
   - Your contribution will be acknowledged in the next publication
4. Only after the CLA is signed may you submit a pull request

---

## Bug Reports

Bug reports are welcome via GitHub Issues. Please include:

- Python version and OS
- Minimal code to reproduce the bug
- Expected vs actual behavior
- Spectral density parameters used

Bug reports do not require a CLA.

---

## Feature Requests

Feature requests are welcome via GitHub Issues. We cannot guarantee implementation, but we read every suggestion.

---
